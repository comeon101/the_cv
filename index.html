CSS

PHP

Ik weet niet wat mijn toekomstige baan wordt, dus ik heb een beetje van alles zonder specialisatie.

ASPIRATION


Node.js

Ik hou ervan hoe Node.js compileert naar low-level code. Het laat JavaScript programmeurs toe om gehele low-level computer programma te ontwikkelen met alleen maar JavaScript. Fantastisch prestatie met high-end scripttaal.

Andere ding over Node.js is dat het event-driven is, dus code is opgestart wanneer nodig anders wacht het. Het bespaart cycles en maakt programmeren gemakkelijker. Er is geen moeite om code schoon te houden.


Neural Processing

Neural Processing is leuk, omdat het proces nooit eindigt. Je kunt altijd proces verbeteren. Om meer accuraat resultaten te krijgen, om het sneller uit te voeren of om meer meta-data te verkrijgen, er is altijd iets om te verbeteren.

Het heeft zijn start, het heeft een doel, maar het is nooit echt klaar. Goed voorbeeld is sprak naar tekst van YouTube of Google Vertaler. Ze waren slecht, ze hadden een doel, en ze worden geüpdate totdat ze bijna perfect zijn.


C#

C# is taal die net hoger staat dan C++, het is simpeler, maar is ook langzamer. Ik heb niet zoveel om erover te vertellen. Het bespaart mensen problemen die het niet nodig hebben. JavaScript en C# blijken meest aangename programeertalen.
